# Moléculas

Moléculas são componentes reutilizáveis e compostos que combinam átomos básicos do design de interface. Na metodologia do Atomic Design, as moléculas são elementos mais complexos, formados pela combinação de átomos (componentes atômicos).

Elas possuem uma função específica e podem incluir elementos como botões, campos de formulário ou barras de navegação. As moléculas são os blocos de construção fundamentais para a criação de organismos (componentes de alto nível) e são projetadas para serem independentes e facilmente utilizáveis em diferentes contextos.
