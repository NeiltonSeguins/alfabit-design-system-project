# Átomos

Os "átomos" são os elementos mais básicos e indivisíveis do design de interface. Eles representam componentes simples, como botões, campos de texto, ícones ou cores específicas. Na metodologia do Atomic Design, os átomos são a base para a criação de componentes mais complexos.

Eles possuem uma única responsabilidade visual ou funcional e são projetados para serem reutilizáveis em todo o sistema. Os átomos podem ser combinados para formar "moléculas", que são componentes mais elaborados compostos por diferentes átomos.

Esses átomos reutilizáveis e independentes permitem um desenvolvimento modular e escalável do front-end, proporcionando consistência e eficiência no processo de design e codificação.
